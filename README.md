# 📊 Sales Performance Dashboard (Excel + VBA)

## 📌 Project Overview
This project showcases an **interactive sales performance dashboard built in Microsoft Excel**, enhanced with **VBA and Macros** for automation and improved user experience.

The dashboard analyzes sales data across multiple regions and sales executives, enabling users to monitor performance against targets and identify key business insights.

---

## 📂 Dataset Description

The dataset consists of sales data for multiple employees across different regions:

- Employee Code & Sales Executive Name  
- Region (Chennai, Delhi, Mumbai, Nagpur, Patna, Pune, Ranchi, Surat)  
- Daily Sales (Day 1 to Day 5)  
- Total Sales  
- Target Sales  
- Target Hit %  
- Away From Target %  

---

## 🛠️ Tools & Technologies

- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Slicers (Region-based filtering)  
- VBA (Visual Basic for Applications)  
- Macros for dashboard control  
- Conditional Formatting  

---

## ⚙️ VBA & Automation Features

- Dashboard navigation using **button-controlled macros (Dashboard 1–4)**  
- Automated updates based on user selections  
- Dynamic filtering using region buttons  
- Reduced manual interaction through automation  

---

## 📊 Dashboard Features

### 🔹 Multi-Dashboard System
- Dashboard 1: Sales performance overview  
- Dashboard 2: Sales comparison by executives  
- Dashboard 3: Target achievement analysis  
- Dashboard 4: Gap analysis (Away from Target %)  

### 🔹 Key Visualizations
- 📈 Bar charts for sales comparison  
- 🥧 Pie chart for target achievement distribution  
- 📉 Line chart for performance trends  
- 📊 KPI tables for quick insights  

### 🔹 Interactive Controls
- Region selection buttons (Chennai, Delhi, Mumbai, etc.)  
- Dashboard navigation buttons  
- Dynamic updates based on user input  

---


## ⚙️ How It Works

- Raw data is structured in tabular format  
- Pivot Tables aggregate sales data  
- Charts dynamically update based on filters  
- VBA macros control:
  - Dashboard switching  
  - Interactive elements  
  - Data refresh  

---

## 🔍 Key Insights

- Identification of top-performing sales executives  
- Regional performance comparison  
- Gap analysis between actual sales and targets  
- Detection of underperforming areas  
- Better visibility into shortfalls using "Away From Target %"
